# Michigan Building Code 2021 (IBC 2021) — Quick Reference

> **Source:** [UpCodes — Michigan Building Code 2021](https://up.codes/viewer/michigan/mi-building-code-2021)
> (merged text: International Building Code 2021 + Michigan amendments). Body text read openly from
> UpCodes (ICC content is ungated, unlike NFPA 70).
>
> ## ⚠️ Scope caveat — this is probably NOT your house's code
> **§101.2:** the Michigan **Building** Code does **not** apply to *detached one- and two-family
> dwellings and townhouses ≤ 3 stories with separate egress* — those are governed by the **Michigan
> Residential Code (IRC)** → see `Michigan-Residential-Code-2021-reference.md`. Use THIS file for: any
> structure larger than a 1–2 family dwelling, the **structural / wood / glazing / foundation /
> roof** engineering backbone (the MRC cross-references the same principles), or accessory/commercial
> buildings on the property. Items also useful for a home are flagged **🏠**.
>
> Units in US customary as the code uses them (mm shown where the code lists them). `psf` = lb/ft²,
> `o.c.` = on center, `AFF` = above finished floor, `FSD` = fire separation distance.

# CHAPTER 1 — Scope & Administration
- **§101.2 Scope / exclusion** 🏠 — applies to all buildings **except detached 1- & 2-family dwellings & townhouses ≤ 3 stories** with separate egress (→ Michigan Residential Code).
- **§105 Permits** — required for construction/alteration/repair/relocation + electrical/mech/plumbing/solar. **Not required** (§105.2): accessory structures **< 120 ft²**, fences **< 7 ft**, painting/finish, minor repairs.
- **§110 Inspections** — footing/foundation, slab, framing, gypsum, firestop, energy, final.
- **§111 Certificate of Occupancy** — required before occupancy/change of use.
- Michigan amendments reference the R 408 rule series and the companion MI Mechanical/Plumbing/Energy/Rehab codes.

# CHAPTER 2 — Definitions
Defines the terms used throughout (occupancy groups, fire separation distance, grade plane, story, mezzanine, fire area, means of egress, occupiable vs habitable space, etc.). Consult when a defined term drives a threshold.

# CHAPTER 3 — Occupancy Classification & Use
| Group | Use |
|---|---|
| **A-1…A-5** | Assembly — theaters (1), food/drink (2), worship/rec/museums/gyms (3), indoor sports (4), outdoor (5). OL < 50 → Group B |
| **B** | Business — offices, clinics, banks, labs, data |
| **E** | Educational — schools through 12th grade; daycare 6+ |
| **F-1 / F-2** | Factory — moderate / low hazard |
| **H-1…H-5** | High hazard — detonation, deflagration, combustion, health, semiconductor |
| **I-1…I-4** | Institutional — custodial, medical, detention, daycare |
| **M** | Mercantile — retail, markets, fuel |
| **🏠 R-1 / R-2 / R-3 / R-4** | Residential — hotels / apartments & dorms / **1–2 dwelling units & care ≤5** / care 6–16 |
| **S-1 / S-2** | Storage — moderate / low (parking garages) |
| **🏠 U** | Utility/Misc — **private garages, carports, sheds, accessory structures** |
- §302.1 mixed/accessory occupancy; §311.1.1 accessory storage classes with the main occupancy.

# CHAPTER 4 — Special Detailed Requirements (by use)
Supplemental rules for: high-rise (403), atriums (404), underground (405), parking garages (406), Group I-2 (407), live/work units, covered/open mall, motor-fuel (406.7), hazardous materials (414/415), spray rooms, stages, special amusement, storm shelters (423 — **ICC 500**), and **§420 Groups R dwelling/sleeping-unit separations**.

# CHAPTER 5 — General Building Heights & Areas
- **Allowable height** ft (Table **504.3**) & **stories** (Table **504.4**) by occupancy × construction type × sprinklers (sprinklers add height/stories). Type I ≈ unlimited; Type V ≈ 40–70 ft.
- **Allowable area** (Table **506.2**): `Aa = At × (1 + If)`; **frontage increase** If (§506.3) needs ≥ 25% perimeter on a public way / open space ≥ 20 ft deep (factor 0–0.75).
- **§503.1** each portion split by a **fire wall** = a separate building for height/area.
- **§505 Mezzanine** 🏠 — ≤ **1/3** of room area (up to 1/2 w/ sprinklers), 7 ft clear above & below, not counted as a story/area.
- **§506.1.3** basement excluded if ≤ one-story allowance. **§507** unlimited-area (sprinklered + 60 ft yard).

# CHAPTER 6 — Types of Construction
- **Type I/II** noncombustible; **III** noncombustible exterior / any interior; **IV** mass/heavy timber (IV-A/B/C + HT); **V** any material (**V-A** 1-hr protected, **V-B** unprotected — *typical wood-frame house if it were under the IBC*).
- **Table 601** fire-resistance ratings (hours):

| Element | I-A | II-A / II-B | III-A / III-B | V-A / V-B |
|---|---|---|---|---|
| Primary structural frame | 3 | 2 / 0 | 1 / 0 | 1 / 0 |
| Bearing wall (exterior) | 3 | 2 / 2 | 2 / 2 | 1 / 0 |
| Floor construction | 2 | 2 / 1 | 1 / 1 | 1 / 0 |
| Roof construction | 1½ | 1 / 0 | 1 / 0 | 1 / 0 |

- **Table 602 / 705.5** exterior-wall rating by **fire separation distance** (closer to lot line → higher rating; FSD ≥ 30 ft → 0 hr for most).

# CHAPTER 7 — Fire & Smoke Protection Features
- **706 Fire walls** — full-height, create separate buildings. **707 Fire barriers** — rated shafts/occupancy/exit separations. **708 Fire partitions** 🏠 — dwelling-unit & corridor separations (**½ hr or 1 hr**). **709 Smoke barriers**. **710 Smoke partitions**.
- **713 Shaft enclosures** — **1 hr** (< 4 stories) / **2 hr** (≥ 4 stories).
- **714 Penetrations / firestops**; **716 Opening protectives** (fire-door ratings, Table 716.1 — e.g., 1-hr wall → ¾-hr door); **717 Fire/smoke dampers**; **718 Fireblocking & draftstopping** 🏠 (concealed spaces, ≤ 1,000 ft² draftstop areas).

# CHAPTER 8 — Interior Finishes
- Wall/ceiling flame-spread classes (ASTM E84 / UL 723): **Class A** 0–25, **Class B** 26–75, **Class C** 76–200; smoke-developed ≤ 450 all.
- **Table 803.13** by occupancy/location — stricter in exit stairs/passageways > corridors > rooms. **Group R** dwellings ≈ **Class C** throughout.
- **§804 Floor finish** — Class I (≥ 0.45 W/cm²) / Class II (≥ 0.22) in exit routes (ASTM E648). Foam plastic prohibited as exposed finish (§803.4).

# CHAPTER 9 — Fire Protection & Life Safety Systems 🏠
- **903.2 Sprinklers required** by occupancy/area — common trigger **fire area > 12,000 ft²**, > 3 stories, or aggregate > 24,000 ft²; **all Group R fire areas** require sprinklers. Standards: **NFPA 13** (full), **13R** (R ≤ 4 stories / ≤ 30 ft), **🏠 13D** (1–2 family & townhouses).
- **905 Standpipes** (height/area thresholds); **906 Portable extinguishers**.
- **907 Fire alarm & detection** — by occupancy; **🏠 smoke alarms** in dwellings (also MRC R314). **🏠 915 Carbon-monoxide detection** required where fuel-burning appliances / attached garages (also MRC R315).

# CHAPTER 10 — Means of Egress 🏠
- **1003.2 Ceiling height** ≥ **7 ft 6 in**; **1003.3** protruding objects keep **80 in** headroom.
- **1004 Occupant load** (Table 1004.5): residential **200 gross ft²**/occ, business 150, assembly (unconc.) 15 net, mercantile 60.
- **1005 Egress width** — stairs **0.3 in/occ** (0.2 sprinklered); other **0.2 in/occ** (0.15 sprinklered). Door encroachment ≤ 7 in open, ≤ 50% any position.
- **1006/1007** number & separation of exits — 2 exits where OL > Table 1006.2.1; separation ≥ ½ diagonal (⅓ if sprinklered).
- **1010 Doors** — clear width **32 in**, leaf ≤ 48 in, height **80 in**.
- **1011 Stairways** — riser ≤ **7 in**, tread ≥ **11 in**, headroom **80 in**, width by occupant load (min 44 in, 36 in if OL < 50).
- **1012 Ramps** — slope ≤ **1:12**, **5 ft** landings.
- **1014 Handrails** — **34–38 in** AFF, both sides, graspable, extensions at top/bottom.
- **1015 Guards** — ≥ **42 in** high; openings reject a **4 in** sphere (4⅜ in some triangular conditions).
- **1017 Travel distance** & **1020 Corridor width** (≥ 36 in / 44 in by occupancy).

# CHAPTER 11 — Accessibility
- **ICC A117.1** is the technical standard. **§1103.2.3 detached 1- & 2-family dwellings are EXEMPT** 🏠.
- **Type A / Type B units** in Group R-2/R-3 multifamily (4+ units → all Type B; ≥ 20 units → ≥ 2% Type A). **Table 1106.2** accessible parking (1 per 25, scaling up; 1 van per 6 accessible).

# CHAPTER 12 — Interior Environment 🏠
- **1208 Room dimensions** — habitable ≥ **7 ft** any dimension; one room ≥ **120 ft²**, others ≥ 70 ft²; ceiling ≥ **7 ft 6 in** (baths/kitchens/laundry/storage 7 ft).
- **1204 Light** — natural glazing ≥ **8%** of floor area (or artificial ≥ 10 fc at 30 in).
- **1202/1203 Ventilation** — natural openable ≥ **4%** of floor area, **or** mechanical.
- **1206 Sound** (multifamily) — **STC ≥ 50** airborne, **IIC ≥ 50** impact between units.
- **1209 Access** — attic **20 × 30 in** (where > 30 in clear height), crawl **18 × 24 in**; **1202.2 attic vent** ≥ 1/150 (1/300 with balanced vents/vapor retarder); crawl vent 1 ft² per 150 ft².

# CHAPTER 13 — Energy Efficiency
Points to the **Michigan Energy Code** (Michigan Uniform Energy Code / IECC 2021 with MI amendments) for envelope R-values, fenestration U-factors, air sealing, lighting. (Separate code — pull if needed.)

# CHAPTER 14 — Exterior Walls 🏠
- **1402/1403** weather-resistant envelope + continuous **water-resistive barrier** (e.g., No. 15 felt / ASTM D226 or equal) behind veneer.
- **Table 1404.2** min covering thickness — vinyl 0.035 in, aluminum 0.019 in, fiber-cement lap 0.25 in, 3-coat stucco 0.625–0.875 in, anchored stone veneer 2 in.
- **1404.4 Flashing** at all penetrations/openings/roof intersections (AAMA 711 self-adhered).
- **1404.3 Vapor retarder** by climate zone (MI is mostly **Zone 5/6** → Class I or II on the warm side; verify against energy code).
- **1404.10.1.3 Adhered veneer clearance** — **4 in** above earth, **2 in** above paving, **½ in** above walking surface.

# CHAPTER 15 — Roof Assemblies 🏠
- **1505 Fire classification** A/B/C (ASTM E108/UL 790).
- **1507 Min slopes**: asphalt shingles **2:12** (double underlayment 2:12–4:12), clay/concrete tile 2½:12, metal panel ¼:12 (standing-seam) / ½:12 (lapped), metal shingles 3:12, slate 4:12, wood shingles 3:12, wood shakes 4:12, mineral roll 1:12.
- **1507.1.2 Ice barrier** 🏠 (cold climates incl. MI) — from eave to ≥ **24 in inside the exterior wall line**.
- **1503 Flashing** min 0.019 in (No. 26 galv.); parapet coping. **1502/1611 Roof drainage** + secondary/overflow scuppers (≥ 4 in).

# CHAPTER 16 — Structural Design 🏠 (the engineering backbone)
- **Live loads (Table 1607.1)**: 1–2 family dwellings **40 psf** (sleeping **30 psf**); uninhabitable attic w/o storage 10 psf; stairs **40 psf** (residential) / 100 psf (other), 300 lb concentrated; passenger garage **40 psf** / 3,000 lb; balconies/decks **1.5× served area**, ≤ 100 psf.
- **Roof live load 20 psf** (§1607.14).
- **Deflection (Table 1604.3)**: floors **L/360** (live) / L/240 (D+L); roofs L/240 (Lr/S) / L/180 (D+L); walls L/120 (wind, flexible finish).
- **1608 Snow** — ground snow **Pg** from maps (MI ~ 20–60+ psf by county; **put Pg on the drawings**); compute pf, Ce, Ct, Is, Cs.
- **1609 Wind** — design speed V from Figs 1609.3 (MI ≈ **107–115 mph** Vult, Risk Cat II; verify by site); exposure B/C/D.
- **1613 Seismic** — MI generally low SDC A/B; map SS, S1, site class.
- **1605 Load combinations** (ASCE 7); **1806** presumptive soil bearing (see Ch 18).

# CHAPTER 17 — Special Inspections & Tests
When a registered design professional must provide **special inspection** (concrete, masonry, steel welding/bolting, soils/piles, sprayed fireproofing, EIFS, seismic/wind for higher categories). Statement of special inspections on the permit docs. Largely commercial.

# CHAPTER 18 — Soils & Foundations 🏠
- **Presumptive bearing (Table 1806.2)**: crystalline bedrock **12,000 psf**, sedimentary rock 4,000, sandy gravel/gravel **3,000**, sand/silty/clayey sand **2,000**, clay/silt **1,500 psf**.
- **1809.5 Frost protection** — footings **below the frost line** (MI ~ **42 in** typical; confirm local depth).
- **1804.4 Grading/drainage** 🏠 — slope away **5%** (6 in drop) over first **10 ft**; swales ≥ 2%; impervious ≥ 2%.
- **1805 Damp/waterproofing** 🏠 — under-slab ≥ **6-mil poly** (6-in laps); foundation-wall dampproofing (bituminous / surface-bond mortar); **waterproofing** where high water table, up to ≥ 12 in above max groundwater.
- **1803.5.3 Expansive soils** (PI ≥ 15 etc.) trigger special design.

# CHAPTER 19 — Concrete
ACI 318 by reference; min compressive strength **f′c** by exposure (often **2,500–4,500 psi**; **3,000+ psi air-entrained** for exterior MI freeze-thaw flatwork), cover, reinforcement, footings/slabs. (Material chapter — consult for any structural concrete.)

# CHAPTER 20 — Aluminum
Aluminum Design Manual / AA standards by reference. Rarely a house item.

# CHAPTER 21 — Masonry
TMS 402/602 by reference — masonry walls, anchored/adhered veneer ties & spacing, grouting, lintels, fireplaces/chimneys (clearances, flue). 🏠 relevant for brick veneer, block foundations, masonry chimneys.

# CHAPTER 22 — Steel
AISC 360 / AISI (cold-formed) by reference — structural steel, steel joists, light-gauge framing, deck. 🏠 relevant if the house uses steel beams/columns (e.g., the AddOn I-beams).

# CHAPTER 23 — Wood 🏠 (the framing chapter)
- **Design methods (§2302/2304)**: ASD, LRFD, **Conventional light-frame (§2308)**, or AWC WFCM (§2309). Use **net (actual) sizes** (§2304.2).
- **§2308 Conventional light-frame** — max **3 stories**; stud spacing ≤ **16 in o.c.** exterior / 24 in interior non-bearing (Table 2308.5.1); stud size/height by table.
- **Cutting/notching** 🏠 — stud notch ≤ **1/6 depth** (≤ 40% bearing-wall stud, ≤ 1.5 in); joist holes ≤ **1/3 depth**, ≥ **2 in** from edges, ≤ 2 in dia; notches only in outer thirds, ≤ 1/6 depth.
- **Fastening (Table 2304.10.2)** — stud-to-plate 3×8d toenail; top-plate lap 16d @ 16 in o.c.; subfloor/sheathing 6d/8d at **6 in edges / 12 in field**.
- **§2304.12 Treated wood** 🏠 — sills on concrete/masonry, wood < **6 in** to exposed earth, posts/beams in contact with ground; moisture content ≤ 19% before cover.
- **§2308.12 Wall bracing** (let-in, WSP/OSB, gypsum); fireblocking/draftstopping per Ch 7.
- Span tables for joists/rafters/headers via §2308 / AWC.

# CHAPTER 24 — Glass & Glazing 🏠
- **§2406.4 Safety glazing (hazardous locations)** — required:
  - **In doors** (2406.4.1) and fixed/sidelite panels.
  - **Within a 24-in arc** of a door edge where bottom < 60 in AFF (2406.4.2).
  - **Large windows** (2406.4.3) when ALL: area > **9 ft²**, bottom < **18 in** AFF, top > **36 in** AFF, walking surface within **36 in** horizontally.
  - **Wet areas** (tubs/showers/saunas/pools) where bottom edge < **60 in** above standing surface (2406.4.5).
  - **At stairs/ramps/landings** within 36 in & < 60 in above walking surface (2406.4.6/.7).
- **2406.3 Identification** — permanent etched/sandblasted manufacturer mark (type, thickness, standard).
- **2406.2 Standards** — CPSC 16 CFR 1201 Cat. II or ANSI Z97.1. **2404** glazing wind-load design (ASTM E1300); **2403.3** support deflection ≤ L/175.

# CHAPTER 25 — Gypsum Board, Panels & Plaster
Fastening/spacing for drywall, water-resistant backing in wet areas (cement board for tile in tubs/showers), shear-wall gypsum values, plaster coats. 🏠 relevant for interior finish & rated assemblies.

# CHAPTER 26 — Plastics
Foam plastic insulation (thermal barrier ≥ **15-min**, e.g., ½-in gypsum; flame ≤ 75 / smoke ≤ 450), light-transmitting plastics, FRP, plastic veneer/skylights. 🏠 thermal-barrier rule matters for spray-foam.

# CHAPTERS 27 / 28 / 29 — Electrical / Mechanical / Plumbing (pointers)
- **27 Electrical** → Michigan Electrical Code (NEC) — see `Michigan-Electrical-Code-2023-reference.md`.
- **28 Mechanical** → Michigan Mechanical Code (IMC).
- **29 Plumbing** → Michigan Plumbing Code (IPC) — see `Michigan-Plumbing-Code-2021-reference.md`; **Table 2902.1** fixture counts.

# CHAPTER 30 — Elevators & Conveying Systems
ASME A17.1 by reference; hoistway/machine-room, shaft venting, fire-service. Residential elevators/lifts (A18.1). Rarely a house item.

# CHAPTER 31 — Special Construction
Membrane/air-supported structures, greenhouses, awnings/canopies, **swimming-pool enclosures & barriers (3109 — barrier ≥ 48 in, ≤ 4-in/2-in gaps)** 🏠, solar/photovoltaic panels on roofs (3111), towers/antennas, automatic vehicular gates.

# CHAPTER 32 — Encroachments Into the Public Right-of-Way
Awnings, marquees, signs, areaways projecting over public property. Municipal coordination.

# CHAPTER 33 — Safeguards During Construction 🏠
Site safety, demolition, **temporary egress/lighting**, **excavation & protection of adjoining property (3307)**, fences/barriers, fire safety during construction.

`https://up.codes/viewer/michigan/mi-building-code-2021/chapter/<N>/<slug>` (N = 1–35). The body text
